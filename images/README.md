# Images

## UA Logo (`images/ua-logo.png`)

Place the United Airlines logo here as **`ua-logo.png`** (PNG or SVG).

- Recommended: transparent background, at least 80×80 px
- The logo appears in the navbar. If the file is missing, the site automatically falls back to the tulip SVG.
- Accepted formats: `.png` or `.svg` (update `src` in index.html if you use `.svg`)
